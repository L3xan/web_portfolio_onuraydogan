# L3xan | Kişisel Portfolyo Web Sitesi 🚀

Modern, minimalist ve tamamen duyarlı (responsive) kişisel portfolyo web sitemin kaynak kodları. Bu proje, hem yazılım (Web & Java) hem de tasarım (UI/UX, Animasyon) alanındaki çalışmalarımı sergilemek amacıyla geliştirilmiştir.

## ✨ Özellikler

- **Modern Tasarım:** Açık tonlar, minimalist arayüz ve profesyonel tipografi (Inter & Playfair Display).
- **Smooth Animations:** Sayfa kaydırıldıkça beliren içerikler ve yetenekleri temsil eden "havada süzülen" (floating) rozetler.
- **Dinamik Proje Modalı:** Projeler sayfasında her bir çalışmanın (Kişisel Bütçe Uygulaması, E-spor Afişleri, Java Projeleri vb.) detaylarını gösteren tam ekran modal yapısı.
- **Tam Duyarlı (Responsive):** Mobil, tablet ve masaüstü cihazlarla %100 uyumlu görünüm.
- **Performanslı:** Dışa bağımlılığı azaltılmış, temiz Vanilla JavaScript ve Tailwind CDN ile hızlı yüklenme süreleri.

## 🛠️ Kullanılan Teknolojiler

- **İskelet:** HTML5
- **Stil & Tasarım:** Tailwind CSS (Özelleştirilmiş renk paleti ve font yapılandırması ile) + Özel CSS Animasyonları
- **İşlevsellik:** Vanilla JavaScript (ES6+)
